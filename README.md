# 🤖 مشروع ذكاء اصطناعي - أبو جمال عبدالناصر الشوكي

- دعم اللغة العربية بشكل كامل
- وضع ليلي/نهاري
- إعادة تعيين المحادثة
- حفظ حقوق المطور
